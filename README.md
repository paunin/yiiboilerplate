Yii Boilerplate App
==========

## Requires

* PHP >= 5.3
* [PHP Composer](https://getcomposer.org/)
* PostgreSQL

## Installation

    #git clone https://github.com/paunin/yiiboilerplate.git yourProject && cd yourProject
    #./protected/install.sh
